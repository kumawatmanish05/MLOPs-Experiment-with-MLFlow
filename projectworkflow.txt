# 🔬 Step 3: Setup MLflow (Local Tracking)

## Start MLflow Server

```bash
mlflow server \
--backend-store-uri sqlite:///mlflow.db \
--default-artifact-root ./mlruns \
--host 127.0.0.1 \
--port 5000
```

## Set Tracking URI in Code

```python
import mlflow
mlflow.set_tracking_uri("http://127.0.0.1:5000")
```

# 🌐 Step 5: Connect with DagsHub

## Initialize DagsHub

```python
import dagshub

dagshub.init(
    repo_owner="your_username",
    repo_name="your_repo",
    mlflow=True
)
```

👉 This automatically:

* Connects MLflow to DagsHub
* Sets remote tracking URI
* Enables online experiment tracking

---

# 🔑 Step 6: Authentication

### Option 1 (Browser Login)

* Automatically opens browser
* Login to DagsHub

### Option 2 (Token - Recommended)

```python
dagshub.init(
    repo_owner="your_username",
    repo_name="your_repo",
    mlflow=True,
    token="YOUR_TOKEN"
)
```
